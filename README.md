# Анализ недвижимости: Data Engineering & ML Pipeline

Полноценный проект по обработке, очистке, хранению и анализу данных о недвижимости (на основе выборки ЦИАН). Проект демонстрирует сквозной пайплайн: от парсинга сырых CSV-файлов и проектирования базы данных до исследовательского анализа (EDA) и обучения модели машинного обучения.

## Стек технологий

* **Языки:** Python, SQL
* **Базы данных:** PostgreSQL 
* **Библиотеки:** `pandas`, `psycopg2`, `scikit-learn`, `matplotlib`, `seaborn`
* **Инфраструктура:** Рекомендуется развертывание БД в изолированных контейнерах (Docker) для удобства тестирования пайплайна.

## Структура проекта

Проект логически разделен на три основных блока: данные, SQL-миграции и исходный код.

### `data/`
* `offers.csv` — исходный датасет с неструктурированной информацией о квартирах (содержит строковые артефакты в ценах, площадях и данных о метро).

### `sql/`
Набор SQL-скриптов для трансформации данных и аналитических выборок:
* `based schema.sql` — базовая проверка загруженных данных в `cian_table`.
* `redacted schema.sql` — скрипт трансформации: очистка адресов, обработка NULL-значений (парковка, потолки) и расчет новой метрики `price_per_meter`.
* `avg price with subway.sql` — аналитический запрос, агрегирующий среднюю стоимость квартир в зависимости от времени в пути до метро.
* `top 10 offers.sql` — поиск 10 самых выгодных предложений (цена за м² ниже средней по рынку при условии пешей доступности метро).

### `src/`
Python-модули для ETL, аналитики и ML:
* `pipeline.py` / `main.py` — основные скрипты ETL-процесса. Читают `offers.csv`, очищают данные с помощью регулярных выражений, создают схему в PostgreSQL и производят bulk-insert.
* `analysis.py` — модуль EDA. Подключается к БД, выгружает данные и строит графики (распределения цен, boxplot для поиска аномалий, корреляционные матрицы).
* `RandomForestRegressor.ipynb` — Jupyter-ноутбук с обучением ML-модели (`RandomForestRegressor`) для предсказания стоимости квартиры на основе площади, удаленности от метро и наличия парковки. Включает оценку по метрикам MAE, R² и кросс-валидацию.
